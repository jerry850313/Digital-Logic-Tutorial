# Digital Logic Design Tutorial System User Guide

## 🌐 Language Selection
You can switch the system language via the navigation bar.

## 📚 Feature Modules

### 1. Course Materials
*   Navigate to the "Handouts" section.
*   Download PDF materials by chapter.

### 2. Exams & Solutions
*   Review previous exam questions and explanations.
*   Click "View Solution" for detailed analysis.

### 3. Verilog Online Simulation
*   Enter Verilog code in the editor.
*   Click "Execute Simulation" to generate waveforms.

### 4. Logic Gate Simulator (Quartus Web)
*   Mimics the Quartus interface.
*   Drag and drop logic gates to build circuits.
*   Supports "Auto Waveform Scan" and "Truth Table".
