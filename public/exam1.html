數位邏輯設計 第一次期中考

第一題：補數之定義與計算 (18%)

請依下面要求並利用補數之定義寫出十進位值 $(37)_{10}$ 之補數值：

數值準備： 十進位值 $N = 37$，位數 $n = 2$。
其二進位為 $(37)_{10} = (100101)_2$ (6位)，八進位為 $(45)_8$ (2位)。
(a) 10 進位的 9 補數：
公式：$(10^n - 1) - N = (10^2 - 1) - 37 = 99 - 37 = \mathbf{62}$
(b) 10 進位的 10 補數：
公式：$10^n - N = 100 - 37 = \mathbf{63}$ (或 9 補數 + 1)
(c) 2 進位的 1 補數：
將 $(100101)_2$ 反轉：$\mathbf{011010}$
(d) 2 進位的 2 補數：
1 補數 + 1：$011010 + 1 = \mathbf{011011}$
(e) 8 進位的 7 補數：
$(45)_8$ 之 7 補數：$(77)_8 - (45)_8 = \mathbf{32_8}$
(f) 8 進位的 8 補數：
7 補數 + 1：$32_8 + 1 = \mathbf{33_8}$

第二題：Canonical and Standard Forms (12%)

(a) $F(x,y,z)=y'+x'y+x'z'$，請寫出 $F$ 之 Canonical Sum of Minterm 形式。

推導過程： 1. 補齊變數：
$y' = y'(x+x')(z+z') = xy'z + xy'z' + x'y'z + x'y'z'$ (m5, m4, m1, m0)
$x'y = x'y(z+z') = x'yz + x'yz'$ (m3, m2)
$x'z' = x'z'(y+y') = x'yz' + x'y'z'$ (m2, m0)
2. 合併重複項並排序：
$F = m0 + m1 + m2 + m3 + m4 + m5$
卡諾圖輔助說明：

答案：$F(x,y,z) = \Sigma(0, 1, 2, 3, 4, 5)$

(b) $F(x,y,z)= xyz+yz'+x'z'$，請寫出函數 $F'$ 之 Canonical Product of Maxterm 形式。

推導過程： 1. 先找 $F$ 的 Minterm：
$xyz = m7$
$yz' = (x+x')yz' = xyz' + x'yz' = m6 + m2$
$x'z' = x'(y+y')z' = x'yz' + x'y'z' = m2 + m0$
所以 $F = \Sigma(0, 2, 6, 7)$
2. $F'$ 的 Minterm 為 $F$ 沒出現的項：
$F' = \Sigma(1, 3, 4, 5)$
卡諾圖輔助說明：

※ 1 為 $F$ 之 minterms；0 為 $F$ 補數項 (即 $F'$ 之 minterms)。

3. 將 $F'$ 的 Minterm 轉為 Maxterm 形式：
Minterm index $i$ 對應的 Maxterm 為 $M_i$

答案：$F'(x,y,z) = \Pi(0, 2, 6, 7)$

第三題：兩級邏輯電路實現 (32%)

請將 $F(A,B,C,D)=\Sigma(1,2,3,5,6,7,8,10)$ 用最簡化兩級邏輯電路實現。

卡諾圖化簡 (4-variable)：
推導基礎式：
SOP 最簡式：$F = A'D + A'C + AB'D'$
POS 最簡式：$F = (A+C+D)(A'+B')(A'+D')$
兩級邏輯實現：
(a) AND-OR
(b) OR-AND
(c) NAND-NAND
(d) NOR-NOR
(e) NOR-OR
(f) NAND-AND
(g) OR-NAND
(h) AND-NOR

第四題：布林代數化簡 (20%)

(a) $F(A,B,C)=\Sigma(0,1,3,4,6)$

卡諾圖化簡 (3-variable)：
卡諾圖圈選推導：
- ● $m_1, m_3$ $\rightarrow A'C$
- ● $m_4, m_6$ $\rightarrow AC'$
- ● $m_0$ 可與 $m_1$ 合併為 $A'B'$，或與 $m_4$ 合併為 $B'C'$

答案：
SOP: (以下任一皆可)
1. $F = A'C + AC' + A'B'$
2. $F = A'C + AC' + B'C'$
POS: $F = (A'+C')(A+B'+C)$

(b) $F(A,B,C,D)=\Sigma(0,1,2,3,7,8,9,11,12)$

卡諾圖化簡 (4-variable)：
卡諾圖圈選推導：
- ● $m_0, m_1, m_3, m_2$ $\rightarrow A'B'$
- ● $m_3, m_7$ $\rightarrow A'CD$
- ● $m_8, m_{12}$ $\rightarrow AC'D'$
- ● $m_9, m_{11}$ $\rightarrow AB'D$

答案：
SOP: $F = A'B' + A'CD + AC'D' + AB'D$
POS: (以下任一皆可)
1. $F = (A+B'+C)(A'+B'+D')(B'+C'+D)(A'+C'+D)$
2. $F = (A+B'+D)(B'+C+D')(A'+B'+C')(A'+C'+D)$
3. $F = (A+B'+C)(A+B'+D)(A'+C'+D)(A'+B'+D')$

(c)第一組 (4-variable)：$F(A,B,C,D)=\Sigma(0,1,2,3,7,8,10,12,13)$

卡諾圖化簡 (4-variable)：
卡諾圖圈選推導：
- ● $m_0, m_1, m_3, m_2$ $\rightarrow A'B'$
- ● $m_{12}, m_{13}$ $\rightarrow ABC'$
- ● $m_0, m_2, m_8, m_{10}$ $\rightarrow B'D'$ (Corners)
- ● $m_3, m_7$ $\rightarrow A'CD$

答案：
SOP: $F = A'B' + ABC' + B'D' + A'CD$
POS: (以下任一皆可)
1. $F = (A+B'+C)(A'+B+D')(B'+C'+D)(A'+B'+C')$
2. $F = (A+B'+C)(A'+B+D')(B'+C'+D)(A'+C'+D')$
3. $F = (A+B'+C)(A'+B+D')(A+B'+D)(A'+B'+C')$

(c)第二組 (5-variable)：$F(A,B,C,D,E)=\Sigma(1,3,6,7,8,9,14,15,16,17,18,20,24,25,28)$

雙層卡諾圖 (5-variable 3D展開)：
五變數卡諾圖拆分為 $A=0$ 與 $A=1$ 兩層平面。
重點圈選說明：
- ● 橘色群：跨層 $m_8,m_9,m_{24},m_{25} \rightarrow BC'D'$ (Essential)
- ● 紫色群：跨層 $m_1,m_9,m_{17},m_{25} \rightarrow C'D'E$ (可用 $AC'D'$ 替換)
- ● 青色群：$A=0, m_1,m_3 \rightarrow A'B'C'E$
- ● 藍色群：$A=0, m_6,m_7,m_{14},m_{15} \rightarrow A'CD$
- ● 紅色群：$A=1, m_{16},m_{20},m_{24},m_{28} \rightarrow AD'E'$
- ● 綠色群：$A=1, m_{16},m_{18} \rightarrow AB'C'E'$

答案：
SOP: (以下任一皆可)
1. $F = A'B'C'E + A'CD + BC'D' + AB'C'E' + AD'E' + C'D'E$
2. $F = A'B'C'E + A'CD + BC'D' + AB'C'E' + AD'E' + AC'D'$
POS: $F = (A+B+C+E)(A+C'+D)(B'+C+D')(A'+C'+D')(A'+C'+E')(A'+D'+E')$

第五題：邏輯電路畫圖 (8%)

$F(x,y,z)=y'+xy'+xz'$，請化至最簡並畫出 NAND 與 NOR 電路。

化簡推導：
$F(x,y,z) = y' + xy' + xz' = y'(1+x) + xz' = \mathbf{y' + xz'}$
卡諾圖分析與化簡：
SOP (Sum of Products) Form:
Group all 1s:
- ● $m_0, m_1, m_4, m_5 \rightarrow y'$
- ● $m_4, m_6 \rightarrow xz'$
$\Rightarrow F = \mathbf{y' + xz'}$

POS (Product of Sums) Form:
Group all 0s:
- $m_2, m_3 \rightarrow (x+y')$
- $m_3, m_7 \rightarrow (y'+z')$
$\Rightarrow F = \mathbf{(x+y')(y'+z')}$
(a) 兩級 NAND-NAND 電路

$F = (y \cdot (xz')')'$

(b) 兩級 NOR-NOR 電路

對偶推導 $F = ((y'+x)' + (y'+z')')' = y'+xz'$

第六題：Parity 系統設計 (10%)

設計 4-bit 偶同位(Even Parity)傳輸系統之：同位產生器G(Parity Generator) 與 檢查器 C(Parity Checker)。

原理說明：
偶同位（Even Parity）確保傳輸的 5 個位元（$A_3,A_2,A_1,A_0,G$）中 '1' 的總數恆為偶數。
這可以透過 XOR 運算來達成，因為 XOR 運算等同於模二加法。
產生器設定 $G = A_3 \oplus A_2 \oplus A_1 \oplus A_0$。
接收端檢查器驗證總和 $C = A_3 \oplus A_2 \oplus A_1 \oplus A_0 \oplus G$。若 $C=0$ 代表無誤。
同位產生器 (Generator G)

$$ G = A_3 \oplus A_2 \oplus A_1 \oplus A_0 $$

同位檢查器 (Checker C)

$$ C = A_3 \oplus A_2 \oplus A_1 \oplus A_0 \oplus G $$
